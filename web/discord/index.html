First-time setup Create your Discord bot About five minutes

Project Akira connects through a Discord bot owned by you. Akira does not operate a shared cloud bot, and your token remains in the Windows credential store on this computer.

Open Discord Developer Portal Discord setup reference
  1. 1

    Create the application and token

    Create an application named Akira. Open Bot, choose Reset Token, and copy the new token. Discord will not show it again unless you reset it.

  2. 2

    Configure installation

    Open Installation, enable Guild Install, and choose Discord Provided Link. Under Guild Install, add the bot and applications.commands scopes, then grant Send Messages.

  3. 3

    Add Akira to Discord

    Copy the installation link, open it in your browser, choose Add to server, and select a private or test server. Open a direct message with Akira from the server member list.

  4. 4

    Copy your Discord user ID

    In Discord, open User Settings → Advanced and enable Developer Mode. Right-click your account and choose Copy User ID.

    Official user-ID instructions
  5. 5

    Connect Project Akira

    Paste the bot token and your user ID below. Enable Start Discord with Akira, save the settings, and wait for Healthy and Ready.

Keep the bot token private. Never paste it into chat, screenshots, commits, issues, or pull requests. Reset it immediately in the Developer Portal if it is ever exposed.
